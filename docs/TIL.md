## TDD에 익숙해지는 과정중
- 블록체인 엔진의 Node를 개발하는 항목에 TDD를 적용하여 개발중이다. 좀 더 익숙해지면 더 빠르게 개발이 가능해질 것 같다.
- 항상 어렵다고 느끼는 부분중 하나인데 코드를 복사하거나 사용하기 위해 이해하는 것은 쉽지 않은 단계다.
  * 그로 인해 좀 더 성숙해지고 단단해지는 것 같아 좋다. 
  * 더 열심히 다른 사람의 코드를 봐야겠다.
